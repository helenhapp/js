Початок роботи з JavaScript

Частина 1: VS Code

✦ Теорія
▶

Що таке VS Code?

VS Code (Visual Studio Code) — це безкоштовний та зручний редактор коду від Microsoft.

Як встановити VS Code:

  1. Перейти на офіційний сайт: VS Code
  2. Вибрати версію для Windows.
  3. Установити за допомогою стандартних налаштувань (Next → Next → Install).

Extensions (розширення)

Extensions — це додаткові модулі, які розширюють можливості VS Code.

Як встановити розширення:

  1. Відкрити VS Code.
  2. У лівій панелі натиснути іконку Extensions (чотири квадратики).
  3. У пошуку ввести назву розширення.
  4. Натиснути Install.

Корисні розширення для веб-розробки:

  • Live Server — запускає сайт з автоперезавантаженням.
  • Prettier – Code Formatter — автоформатування та красивий код.
  • ESLint — підказує помилки у JavaScript.
  • Auto Rename Tag — автоматично змінює закриваючий HTML-тег при зміні відкриваючого.

Auto Save

Auto Save — це функція автоматичного збереження файлів під час роботи. Як увімкнути Auto Save: Меню File → Обрати Auto Save.

✦ Завдання
▶
  1. Встановити VS Code.
  2. Встановити розширення:
    • Live Server
    • Prettier
    • ESLint
    • Auto Rename Tag
  3. Увімкнути Auto Save.

Частина 2: Script

✦ Теорія
▶

HTML Boilerplate

HTML boilerplate — це базовий шаблон сторінки, з якого починається будь-який HTML-файл. У VS Code boilerplate створюється автоматично: введи ! → натисни Tab.

Тег <script>

Тег script дозволяє додати JavaScript до HTML-сторінки. Його можна використовувати двома способами:

  • Писати код прямо в HTML: 
    <script>
  console.log("Script is working!");
</script>
  • Підключити зовнішній файл: 
    <script src="script.js"></script>

Найкраще ставити тег script перед закриваючим тегом </body>, щоб спочатку завантажився весь контент сторінки, а вже потім — JS код.

Файли .js — де писати JavaScript

Усі скрипти зазвичай пишуть у зовнішніх файлах з розширенням .js:

  • script.js
  • app.js
  • main.js

Це робить код охайним, легшим для читання та дозволяє підключати один JS-файл до кількох HTML-сторінок.

Стандартна структура проєкту:

project/
│   index.html
│   script.js
│   style.css
✦ Завдання
▶
  1. Створити папку для проєкту (Тема1).
  2. Створити файл index.html і додати HTML boilerplate.
  3. Створити файл script.js.
  4. В index.html у кінці body підключити JS-файл: 
    <script src="script.js"></script>
  5. Відкрити index.html у браузері за допомогою Live Server.

Частина 3: Console

✦ Теорія
▶

Console (консоль)

Консоль — це спеціальне місце у браузері, де JavaScript показує повідомлення, результати обчислень, попередження та помилки. Тут можна тестувати код “вживу” і спостерігати, як він працює.

Як відкрити консоль:

  1. Відкрий браузер Chrome;
  2. Натисни F12 (або Ctrl + Shift + I);
  3. Перейди у вкладку Console.

console.log()

console.log() — команда, яка виводить текст або числа у консоль. Використовується для перевірки роботи коду та пошуку помилок. 

console.log()

Виведення тексту

Текст пишемо в лапках:

console.log("Hello, world!");
console.log("My name is Olena.");

Виведення чисел

Числа пишемо без лапок — і можемо виконувати з ними будь-які математичні дії: 

console.log(5 + 7);
console.log(10 - 3);
console.log(4 * 2);
console.log(9 / 3);
console.log(10 % 3);   // остача
console.log(2 ** 5);   // степінь
✦ Завдання
▶
  1. Відкрити у браузері консоль.
  2. Виведення тексту у консоль:
    • Вивести текст: "Hello, world!"
    • Вивести ще два свої власні речення
  3. Математика у консолі — обчислити та вивести результат:
    • 5 + 8
    • 12 - 7
    • 4 x 3
    • 20 : 5
    • остача від 17 : 3
    • 62

Залікова робота 1

Гра "Камінь-Ножиці-Папір"

✦ Файли
▶
  1. Створіть папку з назвою "Залік1"
  2. Завантажте і перемістіть в цю папку файли:
    • index.html
    • style.css
  3. Створіть файл script.js і правильно під'єднайте його до index.html
  4. Відкрийте index.html в браузері з допомогою Live Server
✦ Завдання
▶
  1. Створити функцію play(), яка приймає один параметр — вибір гравця (наприклад: "✊ Камінь"). Всередині функції:
    1. Створити масив можливих варіантів гри:
      • ✊ Камінь
      • ✌ Ножиці
      • ✋ Папір
    2. Згенерувати випадковий вибір комп’ютера:
      • Отримати випадковий індекс від 0 до 2
      • Взяти елемент масиву за цим індексом
    3. Сформувати початкове повідомлення: "Ви обрали ... Комп'ютер обрав ..."
    4. Визначити результат гри та додати його до повідомлення. Реалізувати три випадки:
      • Нічия
      • Перемога гравця, якщо:
        • Камінь перемагає Ножиці
        • Ножиці перемагають Папір
        • Папір перемагає Камінь
      • Інакше — перемога комп’ютера
    5. Вивести повідомлення з результатом у HTML: <div id="message"></div>
  2. Для трьох кнопок з id такими як rock, paper та scissors додати event listener, щоб при натисканні викликалася функція play() з відповідним вибором гравця.
✦ Псевдокод
▶ 
функція play(вибір користувача) 
{
  // 1. Створити масив можливих варіантів:
  варіанти = ["✊ Камінь", "✌ Ножиці", "✋ Папір"]

  // 2. Випадково обрати варіант комп’ютера:
  //    - згенерувати випадкове число від 0 до 2 
  //    - знайти елемент з таким індексом в масиві варіантів
  вибір комп'ютера = варіанти[випадкове число від 0 до 2]

  // 3. Підготувати початковий текст результату:
  результат = "Ви обрали playerChoice. Комп'ютер обрав computerChoice."

  // 4. Визначити результат (хто переміг):
  якщо (вибір користувача дорівнює вибору комп'ютера) 
  {
    додати до результату текст " Нічия!";
  } 
  
  інакше якщо 
  (
    (користувач обрав "✊ Камінь" ТА комп'ютер обрав "✌ Ножиці")
    АБО
    (користувач обрав "✌ Ножиці" ТА комп'ютер обрав "✋ Папір") 
    АБО
    (користувач обрав "✋ Папір" ТА комп'ютер обрав "✊ Камінь")
  ) 
  {
    додати до результату текст " Ви перемогли!";
  } 
  
  інакше 
  {
    додати до результату текст " Переміг комп'ютер!";
  }

  // 5. Показати результат на екрані
  додати текст з результатом до елемента з id="message"
}

// 6. Додати обробники кнопок:
взяти елемент з id="rock" і додати event listener "click", який викликає play("✊ Камінь")

взяти елемент з id="scissors" і додати event listener "click", що викликає play("✌ Ножиці")

взяти елемент з id="paper" і додати event listener "click", який викликає play("✋ Папір")

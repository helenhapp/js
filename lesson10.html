Тема 10

Об'єкти

✧ Теорія
▶

Створення об'єкта

const назва_об'єкта = {
  властивість: значення,
  
  назва_методу () {
    // якісь дії
  },
};
const person = {
  // властивості
  name: "John",
  age: 30,
  isStundent: false,
  
  // методи
  sayHello () {
    console.log("Привіт, я " + this.name);
  },
};

Доступ до властивостей та методів об'єкта

console.log(person.name);
console.log(person.age);
console.log(person.isStundent);

person.sayHello();

Зміна властивостей об'єкта та додавання нових

person.age = 12; // змінюємо age
person.isStudent = true; // змінюємо isStudent

person.hobbies = ["reading", "video games"]; // додаємо hobbies

Об'єкти та document.getElementById

const buttons = {
  playButton: document.getElementById("play"),
  restartButton: document.getElementById("reset"),

  editPlayBtn () {
    this.playButton.innerText = "Play";
  },

  editResetBtn () {
    this.restartButton.innerText = "Restart";
    this.restartButton.style.color = "red";
  },
}

buttons.editPlayBtn();
buttons.editResetBtn();
